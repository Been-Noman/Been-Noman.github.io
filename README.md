# Noman Bin Morshed Academic Portfolio

This repository contains a static academic portfolio website for GitHub Pages.

## Live URL after deployment

If the repository is named exactly:

```text
Been-Noman.github.io
```

then the live site should be available at:

```text
https://Been-Noman.github.io/
```

## Files

```text
index.html
assets/css/style.css
assets/js/script.js
assets/docs/Noman_Bin_Morshed_CV.pdf
README.md
```

## How to publish on GitHub Pages

1. Log in to GitHub.
2. Create a new public repository named exactly `Been-Noman.github.io`.
3. Upload all files from this folder into the repository root.
4. Go to repository Settings → Pages.
5. Under Build and deployment, choose source: Deploy from a branch.
6. Select branch: `main`, folder: `/root`, then Save.
7. Visit `https://Been-Noman.github.io/` after GitHub finishes publishing.

## Editing content

- Main website content: edit `index.html`.
- Styling and colors: edit `assets/css/style.css`.
- Mobile navigation behavior: edit `assets/js/script.js`.
- CV file: replace `assets/docs/Noman_Bin_Morshed_CV.pdf` with an updated CV using the same filename.

## Suggested future updates

- Add a profile photo by placing it in `assets/img/` and replacing the NBM placeholder.
- Add Google Scholar, ORCID, and ResearchGate links when available.
- Move manuscripts from “in preparation” to “publications” after submission or acceptance.
